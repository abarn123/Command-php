Cheat Sheet Lengkap Laravel Artisan (Bahasa Indonesia)

Perintah Dasar

Perintah Deskripsi
php artisan make:policy PostPolicy Membuat kelas Policy baru
php artisan --help atau -h Menampilkan bantuan untuk perintah tertentu
php artisan --quiet atau -q Tidak menampilkan pesan apapun
php artisan --version atau -V Menampilkan versi aplikasi
php artisan --no-interaction atau -n Tidak menanyakan pertanyaan interaktif
php artisan --ansi Memaksa output ANSI
php artisan --no-ansi Menonaktifkan output ANSI
php artisan --env Menentukan environment untuk menjalankan perintah
php artisan --verbose (-v, -vv, -vvv) Meningkatkan detail pesan: 1 untuk output normal, 2 lebih detail, dan 3 untuk debug
php artisan clear-compiled Menghapus file kelas yang dikompilasi
php artisan env Menampilkan environment framework saat ini
php artisan help Menampilkan bantuan untuk suatu perintah
php artisan list Menampilkan daftar perintah
php artisan tinker Berinteraksi dengan aplikasi Anda
php artisan down Memasukkan aplikasi ke mode maintenance
php artisan up Mengeluarkan aplikasi dari mode maintenance

Optimasi & Server

Perintah Deskripsi
php artisan optimize [--force] [--psr] Mengoptimalkan framework untuk performa lebih baik --force: Memaksa file kelas dikompilasi --psr: Tidak mengoptimalkan Composer dump-autoload
php artisan serve Menjalankan aplikasi di server pengembangan PHP
php artisan serve --port 8080 Mengubah port default
php artisan serve --host 0.0.0.0 Membuatnya bekerja di luar localhost
php artisan app:name namespace Mengatur namespace aplikasi

Cache & Konfigurasi

Perintah Deskripsi
php artisan auth:clear-resets Menghapus token reset password yang kedaluwarsa
php artisan cache:clear Membersihkan cache aplikasi
php artisan cache:table Membuat migrasi untuk tabel cache database
php artisan config:cache Membuat file cache untuk pemuatan konfigurasi yang lebih cepat
php artisan config:clear Menghapus file cache konfigurasi

Database

Perintah Deskripsi
php artisan db:seed [--class[="..."]] [--database[="..."]] [--force] Mengisi database dengan data --class: Nama class seeder (default: "DatabaseSeeder") --database: Koneksi database yang akan diisi --force: Memaksa operasi di production
php artisan migrate [--database[="..."]] [--force] [--path[="..."]] [--pretend] [--seed] Migrasi database --database: Koneksi database yang digunakan --force: Memaksa operasi di production --path: Path file migrasi --pretend: Hanya tampilkan query SQL --seed: Jalankan seeder setelah migrasi
php artisan migrate:install [--database[="..."]] Membuat repositori migrasi
php artisan migrate:refresh [--database[="..."]] [--force] [--seed] [--seeder[="..."]] Mereset dan menjalankan ulang semua migrasi --seeder: Nama class seeder
php artisan migrate:reset [--database[="..."]] [--force] [--pretend] Rollback semua migrasi database
php artisan migrate:rollback [--database[="..."]] [--force] [--pretend] Rollback migrasi terakhir
php artisan migrate:status Menampilkan status migrasi (up/down)

Pembuatan Komponen

Perintah Deskripsi
php artisan event:generate Membuat event dan handler yang belum ada
php artisan handler:command [-command="..."] name Membuat command handler class baru --command: Command class yang ditangani
php artisan handler:event [--event="..."] [--queued] name Membuat event handler class baru --event: Event class yang ditangani --queued: Handler harus di-queue
php artisan key:generate Mengatur application key
php artisan make:command [-handler] [--queued] name Membuat command baru --handler: Generate handler terpisah --queued: Buat command yang di-queue
php artisan make:console [--command[="..."]] name Membuat Artisan command baru --command: Nama terminal command (default: "command:name")
php artisan make:controller [--plain] name Membuat controller baru --plain: Buat controller kosong
php artisan make:event name Membuat event class baru
php artisan make:middleware name Membuat middleware class baru
php artisan make:migration [--create[="..."]] [--table[="..."]] name Membuat file migrasi --create: Tabel yang akan dibuat --table: Tabel yang akan dimigrasi
php artisan make:model name Membuat Eloquent model class
php artisan make:provider name Membuat service provider class
php artisan make:request name Membuat form request class

Manajemen Queue

Perintah Deskripsi
php artisan queue:table Membuat migrasi untuk tabel queue jobs
php artisan queue:listen [--queue[="..."]] [--delay[="..."]] [--memory[="..."]] [--timeout[="..."]] [--sleep[="..."]] [--tries[="..."]] [connection] Mendengarkan queue tertentu --queue: Queue yang didengarkan --delay: Waktu tunda job gagal (default: 0) --memory: Limit memori (MB, default: 128) --timeout: Timeout (default: 60) --sleep: Waktu tunggu sebelum cek job baru (default: 3) --tries: Percobaan ulang sebelum gagal (default: 0)
php artisan queue:failed Daftar semua failed queue jobs
php artisan queue:failed-table Membuat migrasi untuk tabel failed queue jobs
php artisan queue:flush Menghapus semua failed queue jobs
php artisan queue:forget Menghapus failed queue job tertentu
php artisan queue:restart Restart queue worker
php artisan queue:retry id Mencoba ulang failed queue job (id: ID job)
php artisan queue:work [--queue[="..."]] [--daemon] [--delay[="..."]] [--force] [--memory[="..."]] [--sleep[="..."]] [--tries[="..."]] [connection] Memproses job berikutnya --daemon: Run worker dalam mode daemon --force: Jalankan bahkan dalam maintenance mode

Routing & Lain-lain

Perintah Deskripsi
php artisan route:cache Membuat cache rute untuk registrasi lebih cepat
php artisan route:clear Menghapus cache rute
php artisan route:list Menampilkan semua rute terdaftar
php artisan schedule:run Menjalankan scheduled commands
php artisan session:table Membuat migrasi untuk tabel session
php artisan vendor:publish [--force] [--provider[="..."]] [--tag[="..."]] Publish assets dari vendor packages --force: Timpa file yang ada --provider: Service provider yang memiliki assets --tag: Tag yang memiliki assets
php artisan tail [--path[="..."]] [--lines[="..."]] [connection] Monitor log aplikasi
